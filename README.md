# CET-6 词汇句子练习

大学英语六级词汇句子练习工具。通过「读句子 → 看翻译 → 判断是否掌握」的方式高效自测词汇量，覆盖约 8,000 个六级大纲词汇。

## 如何使用

### 方法一：安装为 App（推荐，手机电脑均支持）

用手机或电脑浏览器打开以下网址（托管在 GitHub Pages）：

```
https://你的用户名.github.io/cet6-practice
```

**安装步骤：**

| 平台 | 操作 |
|------|------|
| iPhone (Safari) | 点击地址栏右侧「分享」→「添加到主屏幕」→ 桌面出现 App 图标 |
| Android (Chrome) | 地址栏下方会自动弹出「添加到主屏幕」，或从菜单选择 |
| 电脑 (Chrome/Edge) | 地址栏右侧出现安装图标，点击即可安装为桌面应用 |

安装后像普通 App 一样使用，有独立图标、无浏览器边框、离线也能用。

### 方法二：直接打开文件

下载后双击 `index.html`，浏览器直接打开。

### 第二步：选择难度

页面顶部有三个等级按钮：

| 等级 | 词汇量 | 说明 |
|------|--------|------|
| 基础级 | ~2,600 词 | 高频常用词汇，适合热身 |
| 进阶级 | ~2,600 词 | 中频词汇，六级核心 |
| 高级 | ~2,600 词 | 低频词汇，冲刺高分 |

建议从「基础级」开始，逐步推进。

### 第三步：开始练习

界面显示 10 条句子，每条句子包含：

```
#42                              ← 词汇序号
The professor emphasized that    ← 英文句子（读这个）
the key to mastering a language
lies in daily practice.

[查看翻译 / 隐藏]                 ← 点击显示中文翻译
教授强调掌握语言的关键在于日常练习  ← 中文翻译（对照检查）

[✓ 认识]   [→ 跳过]              ← 两个按钮
```

### 第四步：判断掌握程度

- **点「认识」**：你看懂了这句话 → 该词标记为已掌握，自动从列表中移除，系统补一句新句子进来。**同时，句中出现的其他六级词汇也会连带标记为掌握**（交叉掌握）。
- **点「跳过」**：你不太确定 → 该句移到列表末尾，稍后再次出现。

列表始终维持 10 条句子。

### 第五步：持续练习

每天打开 `index.html`，继续练习。进度自动保存在浏览器中（localStorage），关闭浏览器、关机重启都不会丢失。

### 备份与恢复

- **备份进度**：点击顶部「备份进度」按钮，下载一个 `.json` 文件。
- **恢复进度**：点击「恢复进度」，选择之前备份的文件即可还原。

### 多版本同步

网页版和桌面版 (.exe) 使用同一份进度文件互通：

```
网页版 → 桌面版：网页版「备份进度」→ 将下载的 .json 文件
                   改名为 cet6_progress.json → 放到 .exe 同目录 → 启动 .exe

桌面版 → 网页版：.exe 同目录下的 cet6_progress.json
                   → 网页版「恢复进度」→ 选择该文件
```

### 多设备同步

由于进度通过文件备份互通，你可以在不同设备间手动同步：

```
电脑上练习 → 备份进度 → 保存到 iCloud/Dropbox 文件夹
                              ↓ 云盘自动同步
手机打开 App → 恢复进度 → 从云盘文件夹选择文件 → 进度同步完成
```

iCloud、OneDrive、Dropbox 都会自动在电脑和手机间同步文件夹，最适合作为中转。

## 常见问题

**Q：进度会丢失吗？**
A：正常情况下不会。进度自动保存在浏览器中。但如果你清除了浏览器数据或使用隐身模式，进度会丢失。建议定期点击「备份进度」以防万一。

**Q：换电脑能用吗？**
A：可以。在原电脑上点击「备份进度」下载文件，在新电脑上打开 `index.html` 后点击「恢复进度」导入。

**Q：需要联网吗？**
A：不需要。所有数据都存在本地文件中，断网也能用。

**Q：需要安装什么吗？**
A：不需要。电脑自带浏览器即可，双击 `index.html` 就能打开。

## 自适应学习策略

系统根据你的学习情况自动调整：

| 词的状态 | 处理方式 |
|----------|----------|
| 从未见过的词 | 优先出现在新一批句子中 |
| 见过但跳过的词（纠结中） | 适当再次出现，加强记忆 |
| 已掌握的词 | 极少再次出现，仅偶尔复习 |

## 获取与更新

### 下载

在 [Releases](https://github.com/你的用户名/仓库名/releases) 页面下载最新版本，或直接克隆仓库：

```bash
git clone https://github.com/你的用户名/仓库名.git
```

下载后双击 `index.html` 即可使用。

### 更新到最新版

如果你是通过 git clone 获取的，在仓库目录下执行：

```bash
git pull origin master
```

如果你下载的是 ZIP 压缩包，重新下载最新版并覆盖旧文件即可。你的学习进度保存在浏览器中，覆盖文件不会丢失。

### 自行构建

词库数据来自 `master_vocabulary.json`。如需重新生成 `data.js` 和 `index_data.js`：

```bash
python -c "
import json
with open('master_vocabulary.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
words = [w for w in data['words'] if w.get('sentence_en') and w['sentence_en'] != '[暂无例句]']
with open('data.js', 'w', encoding='utf-8') as f:
    f.write('window.CET6_DATA = ')
    json.dump(words, f, ensure_ascii=False)
    f.write(';')
print(f'Generated data.js with {len(words)} words')
"
```
