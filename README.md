# CET-6 词汇句子练习

## 快速开始（选一种）

### 方式一：在线版（无需下载）

浏览器打开：
```
https://wangshiyuan-nku.github.io/cet6-practice
```

手机或电脑都能用。想装成 App 的话：

| 平台 | 操作 |
|------|------|
| iPhone (Safari) | 底部「分享」→「添加到主屏幕」 |
| Android (Chrome) | 菜单「添加到主屏幕」 |
| 电脑 (Chrome/Edge) | 地址栏右侧安装图标 →「安装」 |

### 方式二：下载网页版

1. 打开 https://github.com/wangshiyuan-nku/cet6-practice
2. 点绿色 **Code** 按钮 → **Download ZIP**
3. 解压到任意文件夹
4. 双击 `index.html`，浏览器自动打开

### 方式三：桌面版 (.exe)

1. 打开 https://github.com/wangshiyuan-nku/cet6-practice/releases/latest
2. 下载 `CET6_词汇练习.exe`
3. 双击运行，独立窗口打开（无需浏览器）

> **提示**：如果之前用过网页版，可以把网页版「备份进度」得到 `.json` 文件改名为 `cet6_progress.json`，放到 `.exe` 同目录下，进度自动继承。

### 方式四：命令行

```bash
git clone https://github.com/wangshiyuan-nku/cet6-practice.git
cd cet6-practice
# 双击 index.html 或执行 python desktop_app.py
```

## 界面说明

页面顶部有三个难度等级：

| 等级 | 词汇量 | 说明 |
|------|--------|------|
| 基础级 | ~2,600 词 | 高频常用词汇 |
| 进阶级 | ~2,600 词 | 中频词汇，六级核心 |
| 高级 | ~2,600 词 | 低频词汇，冲刺高分 |

每个等级显示 10 条句子：

```
#42                              ← 词汇序号
The professor emphasized that    ← 英文句子
the key to mastering a language
lies in daily practice.

[查看翻译 / 隐藏]                 ← 点击显示中文翻译
教授强调掌握语言的关键在于日常练习  ← 对照检查

[✓ 认识]   [→ 跳过]              ← 判断是否掌握
```

### 按钮说明

| 按钮 | 功能 |
|------|------|
| **认识** | 看懂了 → 标记掌握（含交叉掌握）→ 自动补新句 |
| **跳过** | 不确定 → 移到末尾，稍后再次出现 |
| **随机刷新** | 换一批句子，优先推荐未掌握的词汇 |
| **备份进度** | 下载进度文件 |
| **恢复进度** | 从文件还原进度 |
| **重置进度** | 清除所有记录，从头开始 |

## 进度管理与同步

### 自动保存

每次点「认识」或「跳过」，进度自动保存到浏览器中。关闭重启不丢失。

### 手动备份

定期点「备份进度」下载 `.json` 文件，防止浏览器数据被清除。

### 网页版 ↔ 桌面版同步

```
网页版 → 桌面版：
  网页版「备份进度」→ 下载 .json 文件
  → 改名为 cet6_progress.json
  → 放到 .exe 同目录
  → 启动 .exe

桌面版 → 网页版：
  .exe 同目录的 cet6_progress.json
  → 网页版「恢复进度」
  → 选择该文件
```

### 换设备

旧设备「备份进度」→ 文件传到新设备（U盘/微信/云盘）→ 新设备「恢复进度」。

## 常见问题

**Q：进度会丢失吗？**
A：正常不会，自动保存在浏览器里。清除浏览器数据会丢失，建议定期「备份进度」。

**Q：换电脑怎么继续？**
A：旧电脑「备份进度」下载文件 → 新电脑「恢复进度」。

**Q：需要联网吗？**
A：下载版和桌面版不需要，在线版需要。

**Q：需要安装什么吗？**
A：下载版和桌面版都不需要安装任何东西。在线版有浏览器就行。

**Q：桌面版 (.exe) 报毒怎么办？**
A：这是 Python 打包的独立程序，没有病毒。如被杀毒软件拦截，添加信任即可。

## 自适应学习

| 词的状态 | 处理方式 |
|----------|----------|
| 从未见过 | 优先出现 |
| 见过但跳过 | 适当再次出现 |
| 已掌握 | 极少出现，仅偶尔复习 |
